# 🚀 Smart Career Navigator

Smart Career Navigator is a web application that helps users choose the right career path based on their skills and interests.

## 🔑 Features
- User Login & Signup
- Career Recommendation
- Skill Gap Analysis
- Learning Roadmap
- Interactive Chatbot
- Dashboard with Charts

## 🛠 Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib
